# Daily-Coding-Challenges
# Day 01

## Problem Name
Java Stdin and Stdout I

## Platform
HackerRank

## Difficulty
Easy

## Approach
- Use the `Scanner` class to read three integers from standard input.
- Store each integer in a separate variable.
- Print each integer on a new line using `System.out.println()`.

## Time Complexity
O(1)

## Space Complexity
O(1)
# Day 01

## Problem Name
Print Squares

## Platform
Codechef

## Difficulty
Easy

## Approach
- Use a `for` loop to iterate from 1 to 5.
- Calculate the square of each number using multiplication (`i * i`).
- Print the number and its square on a new line.

## Time Complexity
O(n)  (where n = 5)

## Space Complexity
O(1)
